Importar CSV a Base de Datos





Arrastra y suelta un archivo aquí o haz clic para seleccionarlo.




Instrucciones Importantes

Sobre la llave primaria: La llave primaria será extraída directamente de la base de datos seleccionada, de está manera haciendo uso de la llave primaria se insertarán nuevos elementos o se actualizarán los elementos que tengan una llave primaria igual

Encabezados de las columnas: Los encabezados de las columnas en tu archivo CSV deben coincidir exactamente con los nombres de las columnas en la base de datos para ser importados.

Más columnas en el CSV que en la base de datos: Si tu archivo CSV tiene más columnas que la tabla de la base de datos, solo se importarán las columnas que coincidan con los nombres en la base de datos. Las columnas adicionales serán ignoradas.
